2. Statistics and Probability

Topics:

  • Descriptive vs Inferential Statistics
  • Data Types: Continuous vs Discrete, Nominal vs Ordinal
  • Data Visualization: Histograms, Pie Charts, Bar Charts, Scatter Plots
  • Measures of Central Tendency: Mean, Median, Mode
  • Measures of Dispersion: Variance, Standard Deviation
  • Probability Theory and Normal Distribution
  • Correlation and Covariance
  • Central Limit Theorem
  • Hypothesis Testing: p-value, Confidence Intervals, Type 1 vs Type 2 Errors, Z-test, t-test, ANOVA

Recommended Resources:

  • Harish Garg - Statistics
  • Khan Academy - Statistics

Key Insight:

  • A strong foundation in statistics is essential for a data science career.

5. Machine Learning

Preprocessing Techniques:

  • Handling Missing Values and Outliers
  • Data Normalization and Standardization
  • Encoding Categorical Variables (One-Hot & Label Encoding)
  • Feature Engineering and Selection
  • Train-Test Splitting
  • Cross-Validation Techniques

Model Building:

  • Supervised Learning:
    • Linear Regression, Logistic Regression, Gradient Descent
    • Decision Trees, Random Forest, XGBoost
  • Unsupervised Learning:
